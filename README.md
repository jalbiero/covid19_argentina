# Covid19 in Argentina

Regression estimation for Contagions/Deaths in Argentina

## Requirements

- Make
- Octave

Disclaimer: Tested on Kubuntu 18.04 (Octave 4.2.2) and openSUSE Leap 15.1 (Octave 5.1.0)

## Execution

```console
$ make [run ARGS=[plot | mse]] | last-entries
```
